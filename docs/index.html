AgentSuite v0.4.0

Seven role-specific reasoning agents that turn vague intent into precise operating artifacts. Open-source. MCP-compatible with Codex, Claude Code, and Cowork.

What it does

AgentSuite codifies the operating system around AI generation. Each agent walks a six-stage pipeline (intake → extract → spec → execute → qa → approval), persists 26 artifacts to disk, and promotes approved output into a long-lived _kernel/ that downstream agents consume. The result is a reusable creative-ops or product-ops layer, not a one-off content generator.

Install

pip install agentsuite
# or, no install:
uvx agentsuite-mcp

Quick start (CLI)

export ANTHROPIC_API_KEY=sk-ant-...
agentsuite founder run \
  --business-goal "Launch PatentForgeLocal v1" \
  --project-slug pfl \
  --inputs-dir ./examples/patentforgelocal

Quick start (MCP)

Codex — add to ~/.codex/mcp.toml:

[servers.agentsuite]
command = "uvx"
args = ["agentsuite-mcp"]

[servers.agentsuite.env]
AGENTSUITE_ENABLED_AGENTS = "founder"

Claude Code / Cowork — add to .mcp.json:

{
  "mcpServers": {
    "agentsuite": {
      "command": "uvx",
      "args": ["agentsuite-mcp"],
      "env": {"AGENTSUITE_ENABLED_AGENTS": "founder"}
    }
  }
}
