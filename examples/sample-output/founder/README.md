# Sample output — Founder agent (PatentForgeLocal scenario)

This directory is a **complete, deterministic** Founder run committed to the repo so prospective adopters can browse what AgentSuite produces without installing anything.

- **Generated by:** `tests/golden/test_founder_patentforgelocal.py` equivalent invocation under the deterministic mock LLM (`agentsuite.llm.mock._default_mock_for_cli`).
- **Inputs:** `examples/patentforgelocal/` (project brief + voice sample).
- **Reproducible:** Regenerated by the v0.9.3 sample-output script. Bytes are stable from one run to the next under the mock.

## What's here

- `_state.json` — the run's persisted kernel state.
- `inputs_manifest.json`, `extracted_context.json`, `consistency_report.json`, `cost_summary.json`.
- `qa_report.md`, `qa_scores.json` — the QA stage output.
- 7 founder spec artifacts (`brand-system.md`, `founder-voice-guide.md`, etc.).
- `brief-template-library/` — 8 reusable brief templates for downstream agents.
- `TREE.txt` — directory listing for grep convenience.

This is real Founder output rendered against a fixed prompt template under a deterministic mock — exactly what a live `agentsuite founder run` produces, minus the LLM-specific phrasing on free-text bodies.
